# plcwhiteboard
professional PLC-style whiteboard app for 2026 with these features:
Key Features:

1% margins throughout for maximum canvas space
Small title font (13px) as requested
Multiple whiteboard themes: Light, Dark, Grid Light, Grid Dark, Chalkboard, Blueprint
Drawing tools: Pen and Eraser with adjustable size
6 color options with visual color picker
Touch-optimized for mobile devices
Undo functionality and Clear board
Save drawings as PNG images
Modern glassmorphic UI with a clean 2026 aesthetic
